Substrate-conditioned diffusion for ligand-binding protein generation

### Requirements
- rdkit
- deepspeed
- pytorch
- torch_geometric
- torch_cluster
- torch_scatter
- lightning
- lightning-utilities
- biopython
- hydra-zen
- pandas
- pyrosetta
- pytest
- e3nn

For code-style we use
- pylint
- black
- isort
- darglint
- autoflake

And for visualization
- matplotlib
- seaborn

We also include an `environment.yaml` for your convenience.
